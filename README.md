# Ecommerce
Ecommerce web application using Angular 13  | Node.js | Express.js | sequelize    

## Steps for Project Execution :

##### Client Install
```
cd ecommerce-fronted
npm install --force
ng serve 
```
##### Server Install
```
cd ecommerce-backend
npm install 
npm run dev 
```
